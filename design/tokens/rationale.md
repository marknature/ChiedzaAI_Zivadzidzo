# ChiedzaAI / ZivaDzidzo visual language

Chiedza means light, so the base palette uses warm paper, dawn, and amber rather than generic corporate blue. It should feel like clarity arriving in a classroom, while dark ink anchors data-heavy screens.

ZivaDzidzo has a distinct educational violet accent: reflective and scholarly without competing with the shared ChiedzaAI light palette. Other Ziva modules can retain the family surfaces and glow while choosing their own accent.

Risk is encoded with labelled bands and different lightness as well as hue. Screens must show the text band alongside colour; this avoids a red/green-only interpretation and retains meaning for colour-vision differences.

The compact spacing and two-weight type system are deliberately mobile-first. Use display only for a single score or page moment, heading for hierarchy, and chip-label only inside short status pills.
